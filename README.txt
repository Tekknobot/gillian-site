Gillian Writer Website Prototype
================================

Files included:
- index.html
- styles.css
- script.js

How to preview:
1. Unzip the folder.
2. Open index.html in your browser.

What this is:
- A simple, elegant one-page portfolio concept for a writer.
- Fully responsive for desktop, tablet, and mobile.
- Built with placeholder content so you can show the design first.
